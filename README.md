## How To Run
🎽🎽🎽
### 1. Clone The Repo
`git clone https://github.com/swxft/pet-shop.git`

### 2. Run via Docker
- make sure you've manually turned Docker on locally
- in the cli 👉
`docker-compose up` 

### 3. Use the App
CRUD to your heart's content

### 4. Shut Docker Down
`docker-compose down`
